## 使用

### 包配置

- Maven

```
<dependency>
    <groupId>com.xwc1125</groupId>
    <artifactId>xwc1125-common</artifactId>
    <version>1.0.0</version>
</dependency>
```

- Gradle:

```
dependencies {
	implementation 'com.xwc1125:xwc1125-common:1.0.0'
}
```
